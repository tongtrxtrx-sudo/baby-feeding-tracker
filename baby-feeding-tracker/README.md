# 宝宝成长记录小程序

一个温馨、专业的微信小程序，帮助家长科学追踪宝宝的喂养情况和生长发育。

## 功能特性

### 1. 宝宝档案管理
- 支持添加多个宝宝档案
- 记录宝宝基本信息（昵称、性别、出生日期、出生体重、出生身高）
- 支持切换当前宝宝

### 2. 喂养记录
- 记录每顿奶量（ml）
- 支持母乳/配方奶两种喂养方式
- 记录喂养时间和备注
- 查看今日喂养统计（次数、总奶量、平均奶量）
- 历史喂养记录查看和删除

### 3. 生长记录
- 记录身高（cm）、体重（kg）、头围（cm）
- 体重/身高曲线展示（预留图表接入）
- WHO生长标准百分位参考
- 历史生长记录查看和删除

### 4. 生长评估
- 基于WHO儿童生长标准
- 自动计算百分位（P值）
- 评估生长状态（偏低/偏矮/正常/偏高/过高）
- 个性化喂养和生长建议

### 5. 评估报告
- 今日喂养情况评估
- 最新生长数据评估
- WHO标准范围对比
- 数据统计汇总

## 使用说明

### 1. 准备工作
- 下载并安装微信开发者工具
- 注册微信小程序账号（可选，测试可使用测试号）

### 2. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `baby-feeding-tracker` 目录
4. 填写AppID（可使用测试号）

### 3. 添加图标（可选）
为了完整的用户体验，建议在 `assets/images/` 目录下添加以下图标文件：
- home.png / home-active.png
- feeding.png / feeding-active.png
- growth.png / growth-active.png
- report.png / report-active.png
- profile.png / profile-active.png

建议尺寸：81px × 81px

### 4. 运行项目
点击开发者工具中的"编译"按钮即可预览小程序。

## WHO生长标准

内置0-24个月婴幼儿WHO生长标准数据：
- 体重-for-年龄（男/女）
- 身高-for-年龄（男/女）
- 支持P3、P15、P50、P85、P97百分位

## 数据存储

- 使用微信小程序本地存储（wx.setStorageSync）
- 数据按宝宝ID分别存储
- 支持多宝宝数据隔离

## 技术栈

- 微信小程序原生开发
- 无需第三方框架依赖
- 模块化工具函数设计

## 扩展功能建议

1. **图表展示**：接入ECharts或F2图表库，绘制真实的生长曲线图
2. **数据导出**：支持导出CSV/Excel格式数据
3. **云开发**：接入微信云开发，实现数据云同步
4. **提醒功能**：添加定时喂养提醒
5. **照片记录**：支持上传宝宝照片
6. **疫苗提醒**：添加疫苗接种记录和提醒

## 注意事项

- 本小程序仅供参考，不能替代专业医疗建议
- 如有健康问题，请及时咨询医生
- 数据存储在本地，清除缓存会丢失数据
